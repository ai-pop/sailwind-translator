▌ ШАГ 1. Узнай, какой у тебя билд игры — Mono или IL2CPP

1. Открой папку игры (Steam → ПКМ по Sailwind → Manage → Browse local files)
2. Если в корне есть файл `GameAssembly.dll` — у тебя IL2CPP
3. Если такого файла НЕТ, но есть `Sailwind_Data/Managed/Assembly-CSharp.dll`
   — у тебя Mono (рекомендуемый вариант, проверен на 0.38)

▌ ШАГ 2. Скопируй содержимое нужной папки в папку игры

• Если у тебя MONO  → скопируй ВСЁ из `install_mono/`  в корень игры
• Если у тебя IL2CPP → скопируй ВСЁ из `install_il2cpp/` в корень игры

Структура после копирования должна быть:

  <игра>/
  ├── Sailwind.exe
  ├── winhttp.dll          ← наш (IL2CPP-сборка ставит ещё doorstop)
  ├── doorstop_config.ini  ← только для IL2CPP
  ├── .doorstop_version    ← только для IL2CPP
  ├── dotnet/              ← ТОЛЬКО для IL2CPP
  └── BepInEx/
      ├── core/
      ├── config/
      │   └── AutoTranslatorConfig.ini
      └── plugins/
          ├── SailwindTranslator/
          │   ├── SailwindTranslator.dll
          │   └── translations.json
          ├── XUnity.AutoTranslator/
          └── XUnity.ResourceRedirector/

▌ ШАГ 3. Запусти игру

• Первый запуск IL2CPP будет долгим (1–5 минут) — BepInEx делает
  unhollowing, не закрывай консоль.
• Когда увидишь главное меню — нажми E. Должно появиться окно редактора.
• Переключение языка в игре — клавиша F2.

▌ ШАГ 4. Управление

  E  — открыть/закрыть редактор перевода (править прямо в игре)
  F2 — переключить язык RU/EN на лету

▌ ШАГ 5. Перевод

1. AutoTranslator сам переведёт английские строки через Яндекс/DeepL
   и сложит в `BepInEx/Translation/ru/Text/_AutoGeneratedTranslations.txt`
2. Машинный перевод будет корявым — это нормально.
3. Открываешь E в игре → видишь список → правишь русский → жмёшь Save.
4. Правки сохраняются в `BepInEx/plugins/SailwindTranslator/translations.json`.
5. Можно править translations.json во внешнем редакторе — плагин
   подхватит изменения автоматически (hot-reload).

▌ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

• Лог BepInEx: `BepInEx/LogOutput.log.txt`
• Если в логе нет строки "Sailwind Translator v1.1.0 loaded" — плагин не
  подгрузился. Проверь, что .dll лежит в
  `BepInEx/plugins/SailwindTranslator/`.
• Если в логе есть ошибки Harmony — возможно Sailwind использует
  нестандартный UI. Пришли лог — поправлю патчи.
• Если тексты квадратиками — нет кириллического шрифта. Положи в
  `BepInEx/plugins/SailwindTranslator/` любой .ttf/.otf с кириллицей
  (например NotoSans), плагин попробует использовать его при следующем
  запуске.
