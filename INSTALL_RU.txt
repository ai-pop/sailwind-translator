▌ ШАГ 1. Узнай, какой у тебя билд игры — Mono или IL2CPP

1. Открой папку игры (Steam → ПКМ по Sailwind → Manage → Browse local files)
2. Если в корне есть файл `GameAssembly.dll` — у тебя IL2CPP
3. Если такого файла НЕТ, но есть `Sailwind_Data/Managed/Assembly-CSharp.dll` — у тебя Mono


▌ ШАГ 2. Скопируй содержимое нужной папки в папку игры

• Если у тебя MONO    → скопируй ВСЁ из `install_mono/`    в корень игры
• Если у тебя IL2CPP  → скопируй ВСЁ из `install_il2cpp/`  в корень игры

Структура после копирования должна быть:
  <Sailwind>/
    ├── Sailwind.exe
    ├── winhttp.dll            ← наш
    ├── doorstop_config.ini    ← наш
    ├── .doorstop_version      ← наш
    ├── dotnet/                ← ТОЛЬКО для IL2CPP
    └── BepInEx/
          ├── core/            ← наш
          ├── config/          ← наш
          │     └── AutoTranslatorConfig.ini
          └── plugins/
                ├── SailwindTranslator/
                │     ├── SailwindTranslator.dll
                │     └── translations.json
                ├── XUnity.AutoTranslator/
                └── XUnity.ResourceRedirector/


▌ ШАГ 3. Запусти игру

• Первый запуск IL2CPP будет долгим (1-5 минут) — BepInEx делает unhollowing,
  не закрывай окно консоли.
• Когда увидишь главное меню — нажми F1. Должно появиться окно редактора.


▌ ШАГ 4. Управление

  F1  — окно редактора перевода (править прямо в игре)
  F2  — переключить EN/RU на лету
  F3  — перечитать весь UI (если что-то не обновилось)


▌ ШАГ 5. Перевод

1. AutoTranslator сам переведёт английские строки через Яндекс/DeepL
   и сложит в `BepInEx/Translation/ru/Text/_AutoGeneratedTranslations.txt`
2. Машинный перевод будет корявым — это нормально.
3. Открываешь F1 в игре → видишь список → правишь русский → жмёшь Save.
4. Правки сохраняются в `BepInEx/plugins/SailwindTranslator/translations.json`.
5. Можно править translations.json во внешнем редакторе — плагин подхватит автоматически (hot-reload).


▌ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

• Лог BepInEx: `BepInEx/LogOutput.log.txt`
• Если в логе нет строчки "SailwindTranslator ... loaded" — плагин не подгрузился,
  проверь что .dll лежит в `BepInEx/plugins/SailwindTranslator/`.
• Если в логе есть ошибки Harmony — вероятно Sailwind использует нестандартный UI,
  пришли мне лог — поправлю патчи.
• Если тексты квадратиками — нет кириллического шрифта. Положи в
  `BepInEx/plugins/SailwindTranslator/` любой .ttf/.otf с кириллицей (например NotoSans),
  плагин попытается использовать его при следующем запуске.
